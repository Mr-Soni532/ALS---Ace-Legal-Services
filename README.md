## üè∑Ô∏è API Routes :

#### User Routes :-

```
POST    /signup
POST    /login
POST    /verifyOTP
PATCH   /forgotpassword
GET     /getaUserDataByEmail
GET     /addAppointment

```

#### Lawyer Routes :-

```
POST    /login
PATCH   /forgotpassword
GET     /getaUserDataByEmail
GET     /deleteAppointment

```

